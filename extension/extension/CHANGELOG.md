# AllFormatsReady Extension — Changelog

## v1.0.1 — 2026-07-01
- Added SVG, PSD, TIFF, BMP file upload support
- New output formats: GIF, BMP, TIFF, ICO
- Portal preset chips redesigned to compact pill style
- Before/After size summary in results
- Badge colors for all new formats
- Auto-fill feature: inject converted file directly into portal file input
- Pick which converted file to inject from a compact picker
- Visual overlay highlights all file inputs on the current page
- Toast notification confirms successful injection
- Graceful fallback if portal blocks auto-fill

## v1.0.0 — 2026-06-27
- Initial release
- PDF, JPG, PNG, WebP, HEIC input support
- JPG (5 variants), PNG, WebP, PDF, DOCX output
- 20+ Indian portal auto-detection
- Masked Aadhaar PDF generation
- ZIP download support
