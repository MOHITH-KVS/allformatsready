# AllFormatsReady Chrome Extension

## Setup before loading:
1. Download JSZip: https://cdnjs.cloudflare.com/ajax/libs/jszip/3.10.1/jszip.min.js
2. Save it as `jszip.min.js` in this folder

## Load in Chrome:
1. Open Chrome → go to chrome://extensions
2. Enable "Developer mode" (top right toggle)
3. Click "Load unpacked"
4. Select this entire extension folder
5. Extension is now installed!

## Files:
- manifest.json — extension config
- popup.html — the UI
- popup.js — main logic
- portals.js — portal detection database (20+ portals)
- jszip.min.js — ZIP download library (download separately)
- icons/ — extension icons
